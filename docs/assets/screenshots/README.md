# Screenshots

PNG assets for documentation are listed in [PENDING.md](PENDING.md).

Images are **not committed yet** — capture them from a running demo app and save as `{tag}.png` in this directory.

Example:

```
docs/assets/screenshots/dashboard-overview.png
docs/assets/screenshots/agents-playground.png
```

See the style guide in PENDING.md (1440×900, dark theme).
